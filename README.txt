# CashCount - Interactive Money Calculator

## Description
CashCount is a user-friendly, web-based calculator that helps users count and track combinations of US currency (bills and coins). It features an intuitive interface with visual representations of currency, real-time total calculation, dark/light themes, sound effects, and screenshot capabilities.

## Purpose
Created to simplify the process of counting money for cashiers, teachers, students, and anyone who needs to quickly tally different denominations of US currency. It's particularly useful for training new cashiers or teaching children about money management.

## Features
- Visual currency selection
- Real-time total calculation
- Dark/light theme toggle
- Sound effects with volume control
- Screenshot functionality
- Responsive design for all devices
- Accessible interface

## Technical Requirements
- Modern web browser (Chrome, Firefox, Safari, Edge)
- JavaScript enabled
- Internet connection (for loading currency images and sound effects)

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/cashcount.git
   ```
2. No additional installation required - open index.html in a web browser

## Usage Example
A retail cashier needs to count their drawer at the end of their shift. Instead of manually calculating each denomination, they can:
1. Open CashCount
2. Click + or - for each bill/coin quantity
3. Get an instant total
4. Take a screenshot for documentation
5. Submit the final count

## Why It Was Created
Developed to reduce human error in cash counting, provide a modern alternative to manual calculation, and create an educational tool for teaching currency concepts.
