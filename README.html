Installation

Simply clone the repository and open index.html in your browser:

git clone https://github.com/yourusername/cashcount.git

Usage

1. Open index.html in your web browser

2. Click + or - buttons to adjust currency quantities

3. View real-time total at the bottom

4. Use additional features like theme toggle and screenshots as needed
